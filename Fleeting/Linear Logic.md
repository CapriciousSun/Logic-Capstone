Linear logic is the extension upon [[Static Logic|static logic]]. 

## + Formalism +
In static logic, given some proposition $P$, the statement $P \land P = P$ isn't more true than just $P = P$. In linear logic, a level of dynamism is given by the dynamic conjunction $\otimes$, where $P \otimes P$ doesn't necessarily equal $P$.
